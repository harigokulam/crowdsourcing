CSP - Crowdsourcing Platform

A platform for exploring the power of crowd-sourcing (in combination with gamification)

Download this project as a .zip file Download this project as a tar.gz file

What is it?

Crowd-sourcing is not something new. It has been there for some time. But, can we use this inside an IT organization? This is an attempt to create a platform to explore the power of crowd-sourcing within an organization. The combination of Crowdsourcing and Gamification might even change the way IT Organizations operate.

This platform allows the project owners to publish tasks to a bigger forum. People from any of the units can pick up the tasks and start working.

But, how do we motivate the people from other units to contribute? The answer is 'Gamification'.

Steps

  • Keep the work units as small as possible to start with
  • Allocate points for each of the tasks
  • Based on the review by the task owner, the contributor gets portion of 'task points' for the completed tasks
  • Publish the top 5 contributors details (along with photo) on your dashboard to keep the competition alive.
